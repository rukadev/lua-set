# lua-set
A module for creating sets in lua

## API

Usage Guide

### union

```lua
set:union(set: a, set: b)
```

### intersection

```lua
set:intersect(set: a, set: b)
```

### add

```lua
set:add(key: value)
```

### remove

```lua
set:remove(key: value)
```

### contains

```lua
set:contains(key: value)
```

### get

```lua
set:get()
```
